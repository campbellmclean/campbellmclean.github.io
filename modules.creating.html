SEO details

Meta title

Also known as the "Title Tag," is a concise and descriptive title for a web page. It is one of the most critical SEO elements and appears in the browser's title bar and as the clickable link in search engine results. Search engines use the Meta Title to understand the page's main topic and relevance to user search queries. It should ideally be unique, relevant to the content, and contain relevant keywords to improve the page's ranking in search results.

Meta description

A brief summary or snippet that provides more information about the web page's content. It appears below the Meta Title in search engine results and is intended to entice users to click on the link and visit the page. The Meta Description should be compelling, concise, and contain relevant keywords to attract potential visitors. While it doesn't directly affect search engine rankings, a well-crafted Meta Description can improve the click-through rate (CTR) to the website.

Meta keywords

Is a list of relevant keywords or phrases that describe the web page's content

Schema script

This provides structured data to the search engines and makes your content more readable and increases the likelihood of increasing your ranking in search engines. This is kinda technical, but if you want to give it a go here is a tool to help you generate the script. Just copy and paste the script into this field.
